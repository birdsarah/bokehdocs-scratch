Learning More¶

Hopefully this User Guide gets you started with the basics of using Bokeh.

For detailed information about all modules, classes, models, and objects, consult the refguide. If you are interested in learning how to build and develop Bokeh, or for information about how to create a new language binding, see the Developer Guide.

To see ready-made examples of how you might use Bokeh with your own data, check out the gallery on the main documentation site, as well as the Bokeh NBViewer Gallery.

For questions and technical assistance, come join the Bokeh mailing list.

Visit the Bokeh GitHub repository and try the examples.

Be sure to follow us on Twitter @bokehplots, as well as on Vine, and Youtube!
